Kebijakan Privasi

YouTube Auto Livestreaming Pro

Terakhir diperbarui: 31 Mei 2025

Strim (YouTube Auto Livestreaming Pro) berkomitmen untuk melindungi privasi Anda. Kebijakan Privasi ini menjelaskan praktik kami terkait penggunaan aplikasi dalam menangani data dan informasi Anda sebagai pengguna.

Pengumpulan Data

Kami TIDAK MENGUMPULKAN ATAU MENYIMPAN:

  • Informasi identitas pribadi
  • Data otentikasi (stream key sepenuhnya disimpan seara lokal)
  • Informasi keuangan
  • Informasi kesehatan
  • Data lokasi
  • Riwayat streaming
  • Komunikasi pribadi
  • Informasi channel

Penyimpanan Lokal

Data sepenuhnya hanya disimpan secara lokal di dalam device Anda sebagai pengguna:

  • Stream key (disimpan aman di local device)
  • Pengaturan dan preferensi penjadwalan
  • Penggunaan media dan file (video, thumbnail, judul, deskripsi)

Bagaimana kami menangani data Anda:

  • Semua pemrosesan dilakukan secara lokal di device Anda
  • Tidak ada data pengguna yang dikirim ke server kami
  • Tidak ada pembagian data ke pihak ketiga
  • Tidak ada pelacakan atau analitik

Komunikasi Server

Aplikasi hanya berkomunikasi dengan server untuk:

  • Mengambil data video untuk ditampilkan di Youtube
  • Memeriksa ketersediaan update apps
  • Memvalidasi dan menggunakan Youtube API
